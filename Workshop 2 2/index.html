Una metodología para el desarrollo de software se refiere a un framework (marco de trabajo) que es usado para estructurar, planear y controlar el proceso de desarrollo en sistemas de información. A lo largo del tiempo, se han desarrollado muchos métodos, cada uno con sus fortalezas y debilidades Un framework para metodología de desarrollo de software consiste en: Una filosofía de desarrollo de programas de computación con el enfoque del proceso de desarrollo de software Herramientas, modelos y métodos para asistir al proceso de desarrollo de software Estos frameworks se asocian habitualmente con organizaciones que desarrollan, apoyan el uso y promueven la metodología
